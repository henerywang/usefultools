# 前两道填空题常规套路：
* 一个有10个指针的数组，该指针是指向一个整型数的
> int *i[10]  
* 一个指向有10个整型数数组的指针
> int *(i)[10]  
* 一个指向函数的指针，该函数有一个整型参数并返回一个整型数
> int *(i)(int)  
* 一个有10个指针的数组，该指针指向一个函数，该函数有一个整型参数并返回一个整型数
>  int *(i[10])(int)
